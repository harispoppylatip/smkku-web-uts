Program Keahlian

Program keahlian kami dirancang bersama mitra industri sehingga materi pembelajaran selalu relevan dan aplikatif. Setiap program dilengkapi dengan laboratorium, praktik kerja lapangan, dan sertifikasi profesi.

Pilihan Program

Teknik Komputer dan Jaringan

Membahas jaringan komputer, keamanan siber, dan administrasi server untuk memenuhi kebutuhan dunia IT.

Rekayasa Perangkat Lunak

Fokus pada pengembangan aplikasi, manajemen proyek, dan jaminan kualitas perangkat lunak.

Multimedia

Mengasah kreativitas visual melalui desain grafis, animasi, dan produksi konten digital.

Akuntansi

Memahami akuntansi manual dan digital, audit, serta analisis keuangan untuk bisnis modern.

Bisnis Daring dan Pemasaran

Strategi pemasaran digital, pengelolaan e-commerce, dan branding untuk wirausaha muda.

Keunggulan Program

Mitra Industri

Kolaborasi dengan 40+ perusahaan untuk praktik kerja lapangan dan rekrutmen langsung.

Sertifikasi Kompetensi

Siswa difasilitasi mengikuti sertifikasi BNSP dan vendor IT internasional.

Project Based Learning

Setiap semester siswa menyelesaikan proyek nyata sebagai portofolio profesional.
