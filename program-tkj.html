Teknik Komputer dan Jaringan

Program TKJ mempersiapkan siswa menjadi teknisi jaringan, administrator sistem, dan spesialis infrastruktur TI yang siap bekerja maupun menjadi wirausaha mandiri.

Kompetensi Utama

Manajemen Jaringan

Perencanaan, instalasi, dan troubleshooting jaringan LAN/WAN menggunakan perangkat terkini.

Keamanan Siber

Pengenalan sistem keamanan jaringan, firewall, monitoring trafik, dan mitigasi serangan.

Administrasi Server

Pengelolaan server Linux/Windows, virtualisasi, dan layanan cloud untuk kebutuhan bisnis.

Laboratorium & Kegiatan

Siswa belajar di laboratorium jaringan dengan topologi real, melakukan simulasi menggunakan Cisco Packet Tracer, dan mengikuti kompetisi LKS bidang IT Network System Administration.

  • Praktik konfigurasi router, switch, dan wireless access point.
  • Magang di perusahaan provider internet dan instansi pemerintah.
  • Program sertifikasi internasional seperti MTCNA dan CompTIA Network+.
